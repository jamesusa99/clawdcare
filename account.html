Subscriptions

Programs are monthly guided subscriptions from the Shop—coaching, tracks, and longevity workflows (not per-token metering). Each active line bills until you cancel it.

Token Plan is billed monthly like Programs. Each row in the table below is its own renewal; you can hold several plans and add another in the Shop if you run through tokens before the next cycle. Your running balance is in Token balance (below the tables). Tokens power on-demand AI, console skills, and automations (separate from Program fees). Use Edit / Cancel the same way as Programs.
